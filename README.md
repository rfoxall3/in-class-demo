# Enter the Boy Aquarium 🐠🦈🏒

### an unserious web development project

Discover the 2025-26 San Jose Sharks roster through the magic of a fun little webpage. [See it live on Github Pages!](https://rfoxall3.github.io/boy-aquarium/)

## Features
- **Flip:** Interactable boxes with player images on the front and basic information on the back.
- **Sort:** Discover which players have scored the most or played the most games.
- **Learn:** New to hockey? Use the handy infobox to learn what the statistics mean.

## Files
- `index.html`: main HTML file,  defines the structure of the page and where the infobox content lives
- `styles.css`: stylesheet, where the flipbox feature lives and defines the overall styling for the page & features
- `script.js`: Javascript file, holds the information for each player and fills out the flipboxes based on the sort function when clicked
